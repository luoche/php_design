组合模式实例

例子 ： 魔兽游戏中

有不同的人种别

如 种类1 ： 猎手
种类2 ： 火枪手


可以用不同的种类，建立一个陆军军队

可以用不同的种类，建立一个海军军队


可以结合 不同的军队，建立一个超级 海军陆军连队


此外 可以加陆军军队中加入 不同的种类的人
其他依然

视频地址 ： http://edu.51cto.com/lesson/id-35834.html

如果是普通的抽象继承，处理陆军连队和海军连队 方法都是鳙于的


最终是为了

$xxx = new Kind();

$xxx->addKind($_swordman);
$xxx->addKind($_gunner);
$xxx->addKind($_army);


// 可以通过kind 中添加所有的类型 来组合
